# 💬 Voice of Customer AI Agent

## Overview

A Product Management AI dashboard that transforms customer reviews into actionable product insights using AI.

---

## Problem Statement

Product teams spend hours manually reviewing customer feedback to identify pain points and prioritize improvements. This project automates that workflow using AI.

---

## Features

- Upload CSV/Excel reviews
- Sentiment summary
- Issue categorization
- Business priority scoring
- Feature request extraction
- AI-generated user stories
- AI Product Manager recommendations
- Export user stories

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Ollama
- Llama 3.2
